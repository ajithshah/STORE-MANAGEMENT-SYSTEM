## MERN

A Simple MERN Stack project (MongoDB, Express, React, NodeJS)
STEP-1
    Install npm packages(express, body-parser, cors, mongoose).
    Add your mongoDB Atlas url in server.js
STEP-2
    start client    -> cd client
                    -> cd src
                    -> npm start
STEP-3
    start server    -> cd server
                    -> node server.js
